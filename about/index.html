The TXT firmware community edition

This is the community edition of the FischerTechnik TXT firmware.
All source code can be found in the ftCommunity repository at github.

Licenses

This firmware consists of various open source projects distributed under different open source licenses. These include:
  • GPL: Most code comes under this license
  • LGPL: Most libraries are use this license
  • MIT: Another popular license
Please refer to the ftCommunity repository about the licensing of specific packages.
